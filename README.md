---------------------------------------------------------
### 查找重复的文献和对应重复次数

```
$ python find_repeat.py *.bib
```

---------------------------------------------------------
### 读入含重复文献的bib,并写入新的bib

```
$ python delete_repeat.py old.bib new.bib
```
